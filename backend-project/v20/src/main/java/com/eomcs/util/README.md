# LinkedList 만들기

- 1) 최소 클래스를 작성한다.
- 2) 값과 다음 값의 주소를 저장할 Node 클래스를 정의한다.
- 3) LinkedList의 첫 번재 노드와 마지막 노드의 주소, 크기를 저장할 인스턴스 필드를 선언한다. 
- 4) add(Object) 추가
- 5) get(int) 추가 (꺼내는 것)
- 6) insert(int, Object) 추가
- 7) remove(int) 추가
- 8) set(int, Object) 추가 (바꾸는 것)
- 9) toArray() 추가 (목록을 배열로 만들어서 리턴해 주는)
- 10) 제네릭 적용   
- 11) toArray(E[]) 추가 
- 12) size() 추가
